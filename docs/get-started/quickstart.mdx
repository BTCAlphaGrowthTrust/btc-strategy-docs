# Quickstart

Your first cross-strategy correlation in **three calls**, **no key** — these are the open
**preview** tier. New here? Start with **[Start here](/docs/get-started/start-here)** (try-it +
the evaluate-against-your-book recipe). Base URL: `https://btc-strategy-data-api.fly.dev`.

:::note Factual data, not advice
Every response is **backtested / modelled historical** data for professional & institutional
use. Past performance is not indicative of future results — not advice, not a recommendation,
not a forecast.
:::

## 1 · Discover the strategies

```bash
curl https://btc-strategy-data-api.fly.dev/v1/strategies
```

Every response is the canonical envelope — `data`, `meta`, `links`:

```json
{
  "data": [
    {
      "id": "helios",
      "name": "Helios",
      "tv_status": "provisional",
      "verification_state": "dual_verified",
      "base_risk_pct": 0.025,
      "headline": {
        "cagr_pct": 50.51,
        "max_drawdown_pct": -33.32,
        "sharpe_daily_annualized": 1.18,
        "profit_factor": 2.60,
        "trade_count": 240
      }
    }
  ],
  "meta": { "basis": "backtested", "dataset_version": "2026-06-24T07:40:35Z", "count": 14 },
  "links": { "self": "/v1/strategies", "summary": "/v1/summary", "dataset": "/v1/dataset" }
}
```

**Notice — two confidence signals on every strategy:** `verification_state` is our **internal**
validation depth (here `dual_verified` — confirmed two independent ways in-house), and `tv_status`
is the **external** TradingView cross-check (`provisional` until reconciled; all 14 are provisional
today). Two complementary axes, not a contradiction → [Verification & data confidence](/docs/concepts/verification).

## 2 · Pull the correlation matrix

```bash
curl "https://btc-strategy-data-api.fly.dev/v1/correlation?period=daily"
```

Pearson correlation of aligned daily returns — **risk-invariant** (unchanged by `risk_pct`).

**Notice:** pairs span from **atlas / hyperion at +0.93** down to **helios / kronos at −0.09** —
that spread, from tightly-correlated to *negatively*-correlated, is the raw material for diversification.

## 3 · The diversification snapshot

```bash
curl https://btc-strategy-data-api.fly.dev/v1/summary
```

Preview returns the **diversification snapshot** — mean correlation and effective bets. The
equal-weight **book performance** is full-tier (keyed); preview returns it as `null`:

```json
{
  "data": {
    "strategy_count": 14,
    "coverage": { "from": "2018-08-14", "to": "2026-06-25" },
    "mean_pairwise_correlation": 0.353,
    "diversification": {
      "most_correlated":  { "pair": ["atlas", "hyperion"], "correlation": 0.927 },
      "least_correlated": { "pair": ["helios", "kronos"],  "correlation": -0.094 },
      "effective_bets": 2.51
    },
    "equal_weight_book": null,
    "book_level_gated": true
  }
}
```

**What to notice:** the least-correlated pair — **helios / kronos** — sits at **−0.09**, a genuinely
independent, even mildly *hedging*, return stream. Over ~**8 years** (2018 → 2026) the 14 strategies
behave like just **2.5 independent bets** (`effective_bets`) at a mean correlation of **0.35**.
That's the diversification you're buying. *(Backtested / modelled, provisional — not advice.)*
