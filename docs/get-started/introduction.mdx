# Introduction

The **BTC Alpha Strategy Data API** serves **backtested / modelled historical** performance
data for a curated set of BTC trading strategies, over one clean **REST / JSON** interface, for
**professional and institutional** use under a global license.

:::warning Factual historical data — not advice
Everything here is **factual historical data**: backtested/modelled results computed from public
market data. **Past performance is not indicative of future results.** It is **not investment
advice, not a recommendation, not a forecast, and not an offer.** See
[Factual Data, Not Advice](/docs/legal/not-advice).
:::

## At a glance

- **17 validated strategies** on **`BTC-PERPETUAL`** (Deribit; inverse, BTC-settled).
- **~7.8 years** of history (2018-08 → 2026-06), daily and weekly.
- Per-strategy **stats**, **returns**, **equity curve**, plus **cross-strategy correlation** and
  a **book-level summary**.
- **Modelled risk rescaling** via a single `risk_pct` parameter.
- **Open access today** — no key required (a subscription gate is pre-wired for later).

## A note on what this is — and isn't

This is a **data product**, not a financial product. It reports **what these strategies did** in
a backtest over public history, with the methodology fully documented (including its
[limitations](/docs/methodology/overview)). It does not tell you what to trade, and it makes no
claim about future performance.
